# Washing Machine Not Starting

## Overview
Washing machine doesn't power on or doesn't respond when cycle buttons pressed. May have power but motor doesn't start, or no response to any input. Caused by power supply failure, faulty start switch, tripped door lock, or electrical component failure.

## Symptoms
- No power lights or display visible
- Power lights on but buttons don't respond
- Unit hums but drum doesn't rotate
- Cycle buttons don't initiate wash
- Door lock light red (won't unlock)
- Machine powered but non-responsive
- Error code displayed (E1, E2, etc.)

## Possible Causes
**Power Supply Failure**: Main breaker tripped, outlet not functional, or internal power supply board failed.

**Faulty Start Switch**: Mechanical switch doesn't close circuit when pressed.

**Door Lock Not Engaging**: Safety mechanism prevents wash if door not locked. Lock solenoid failed.

**Tripped Circuit Breaker**: Excessive current draw trips breaker. Breaker in OFF position.

**Faulty Capacitor**: Start capacitor doesn't energize motor start coil.

**Thermal Overload**: Motor overheated and safety switch engaged. Prevents operation until cooled.

**PCB Control Board Failure**: Main control board malfunctioning. Cannot process input or activate motor.

## DIY Troubleshooting Steps

1. **Check power**: Is machine plugged in? Test outlet with lamp. Does lamp turn on? If not, outlet is dead.

2. **Check breaker**: Look at home electrical panel. Is washing machine breaker in ON position? If OFF or tripped, reset by toggling OFF then ON. If trips immediately after reset, electrical fault present.

3. **Check door**: Is door fully closed? Many machines won't start if door ajar. Close firmly and listen for lock click.

4. **Try different outlet**: If possible, plug machine into different outlet (different breaker). Does it power on? If yes, original outlet/circuit has issue.

5. **Wait if overheated**: If machine just completed hot water cycle, motor may have triggered thermal overload. Wait 30 minutes. Try again.

6. **Check for error code**: Look at display. Any error code (E1, E3, F1, etc.)? Note it. Error codes indicate specific issues.

7. **Try manual reset**: Some machines have reset button (usually red, on back). Try pressing reset. Wait 2 minutes. Try starting again.

8. **Listen for sounds**: When pressing start, does machine make any sound (hum, click, beep)? No sound = no power. Click = power present but motor not starting.

## Safety Warnings
**Electrical Hazard**: Never reset breaker more than 3 times if it keeps tripping. Indicates serious electrical fault. Call electrician.

**Entanglement**: Water still in drum may release when servicing. Don't reach into drum.

## When To Call A Technician
- Breaker keeps tripping when machine turned on (electrical fault)
- No power despite confirmed working outlet
- Door lock light stays red despite door closed (lock solenoid failure)
- Power present but machine doesn't respond to any button (PCB failure)
- Thermal overload repeatedly triggers (motor issue)
- Error code persists after reset

## Preventive Maintenance
- Use dedicated breaker for washing machine (not shared)
- Install voltage stabilizer (minimum 2 kVA)
- Don't overload machine (check manual capacity)
- Ensure door seals properly before starting
- Allow motor to cool after hot-water cycles before running again
- Check power cord yearly for damage/fraying
