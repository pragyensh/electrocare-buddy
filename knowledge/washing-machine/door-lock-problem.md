# Washing Machine Door Lock Problem

## Overview
Door cannot open or close during/after cycle. Door locked permanently or won't lock during operation. Caused by faulty solenoid lock, mechanical jam, or safety sensor malfunction. Prevents access to clothes or cycle start.

## Symptoms
- Door won't unlock after cycle completes
- Door won't lock to allow cycle start
- Door lock indicator light red/blinking
- Solenoid making unusual sounds
- Mechanical handle stuck
- Emergency release required to access clothes
- Error code E6 or lock-related code

## Possible Causes
**Faulty Solenoid Lock**: Electronic lock doesn't disengage after cycle or won't engage during cycle.

**Mechanical Jam**: Latch or catch mechanism stuck or misaligned.

**Safety Sensor Malfunction**: Sensor doesn't detect door closed. Lock remains engaged as safety measure.

**Door Strike Misalignment**: Door frame latch point misaligned. Door won't seat properly.

**Failed Capacitor": Lock solenoid powered by capacitor. Failed capacitor prevents lock engagement.

**Thermal Overload": Safety system locked door due to motor overheat.

## DIY Troubleshooting Steps

1. **Try emergency release": Most machines have emergency release near lock. Look for small red button or pull handle near lock area. Pull or press to manually release lock. Does door open? If yes, solenoid likely faulty.

2. **Wait for cool down": If machine recently completed hot cycle, thermal overload may have engaged lock. Wait 30 minutes. Try opening again. If door opens, thermal issue (normal response).

3. **Power cycle": Unplug machine from outlet. Wait 30 seconds. Plug back in. Try door. Electronic locks sometimes respond to reset.

4. **Check door alignment": When door is open, look at frame edges. Does door frame line up evenly with machine body? Misalignment prevents proper seating.

5. **Inspect door latch": Inside machine at door frame, look for mechanical catch/latch. Is it visibly bent or out of position? Should be straight.

6. **Examine solenoid": Near lock, look for small solenoid coil (cylindrical, electric-looking component). Any burn marks, loose connections, or heat discoloration?

7. **Try manual cycle": Some machines have manual mode. If available, try starting manual cycle. Does lock respond?

## Safety Warnings
**Electrical": Door lock solenoid operates at 230V. Do not touch solenoid coil while powered.

**Heat": Thermal overload means motor is very hot. Wait fully before touching machine.

**Pressure": Never force door open. Internal pressure may exist (air trapped).

## When To Call A Technician
- Emergency release doesn't work
- Door remains locked despite power cycle
- Solenoid making buzzing/humming continuously (stuck)
- Door frame visibly bent or misaligned
- Safety sensor not responding
- Mechanical latch broken

## Preventive Maintenance
- Don't force door while closed (stress lock)
- Ensure door fully opens after release (prevents latch wear)
- Keep door seal clean (debris prevents proper closure)
- Don't slam door closed
- Have annual inspection of lock mechanism
- Test lock operation monthly (proper engagement/disengagement)
