# 🤖 ElectroCare AI

### AI-Powered Voice Support Assistant for Home Appliances

ElectroCare AI is a modern voice-enabled customer support assistant designed to help users diagnose and troubleshoot common home appliance problems using natural language conversations.

The system combines Speech Recognition, Retrieval-Augmented Generation (RAG), Large Language Models, and Text-to-Speech technology to create an interactive support experience similar to speaking with a real technician.

---

## 🚀 Features

### 🎙️ Voice-First Experience

* Talk naturally using your microphone
* Real-time speech recognition
* Hands-free troubleshooting support
* One-click voice interaction

### 🧠 AI-Powered Troubleshooting

* Intelligent diagnosis of appliance issues
* Context-aware responses
* Conversational problem solving
* Step-by-step guidance

### 📚 Retrieval-Augmented Generation (RAG)

* Uses a custom appliance knowledge base
* Retrieves relevant troubleshooting documents
* Provides accurate and grounded responses
* Reduces hallucinations

### 🔊 Smart Voice Responses

* Supports Browser TTS
* Supports Sarvam AI TTS
* Natural conversational responses
* Voice interruption and stop controls

### 🌐 Bilingual Support

* English Support
* Hindi / Hinglish Support
* Automatic language-aware responses

### 🤖 Interactive AI Assistant

* Animated AI Robot Companion
* Different visual states:

  * Idle
  * Listening
  * Thinking
  * Speaking
* Floating assistant interface

### 🎨 Modern UI/UX

* Futuristic cyber-themed design
* Glassmorphism effects
* Neon glow animations
* Responsive layout
* Smooth interactions

## ⚡ How It Works

1. User speaks or types a problem
2. Speech is converted into text
3. Relevant troubleshooting documents are retrieved
4. AI generates a context-aware response
5. Response is converted back into speech
6. Animated assistant visually reacts to the conversation

## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/electrocare-ai.git
cd electrocare-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
DEEPGRAM_API_KEY=your_deepgram_api_key
SARVAM_API_KEY=your_sarvam_api_key
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:3000
```
